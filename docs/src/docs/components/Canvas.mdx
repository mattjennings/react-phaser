import { Playground, Props } from 'docz'
import { Game, Scene, Canvas } from 'react-phaser-fiber'

The Canvas component will let you customize the canvas element that Phaser renders into. You must be using `react-dom` to use this component.
If you are not using react-dom, you can setup a custom canvas in the [Phaser.Game config](https://photonstorm.github.io/phaser3-docs/Phaser.Game.html#canvas__anchor)

<Playground customCanvas>
  <Canvas style={{ border: '1px solid red' }}>
    <Game width={400} height={400} />
  </Canvas>
</Playground>

## Props

Same props as a regular `<canvas />`
